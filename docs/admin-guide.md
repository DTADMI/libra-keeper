# Admin Documentation

As an Administrator, you have full control over the LibraKeeper system.

## Library Management

- **Add Items**: Use the "Add New Item" button on the dashboard. Provide details like Title, Author, Type, and Cover Image.
- **Edit/Delete**: You can modify or remove any item from its details page.
- **Manage Categories**: Use tags to organize your collection.

## Lending Management

- **Requests**: Go to the "Admin Requests" page to see pending borrow requests.
- **Approve/Reject**: Review requests and set due dates for approvals.
- **Returns**: When an item is returned, mark it as "Returned" to make it available again.

## User Management

- **Roles**: Promote regular users to Admins from the "User Management" page.
- **Activity**: Monitor recent actions in the dashboard's "Activity Feed".

## Data Management

- **Export**: Download your entire collection in CSV or JSON format from the "Admin Settings" page.
- **Suggestions**: Review user suggestions and mark them as processing or completed.

## Communication

- **Messages**: Reply to user inquiries in the "Messages" section.
- **Notifications**: The system automatically sends emails for key events (using Resend).
